# Employee-Layoff-Data-Analysis
Power Bi Dashboard created from a Employess Layoff Data set

•	Analyzed 10,000+ BMW sales records using Python (Pandas, NumPy, Matplotlib, Seaborn) to identify key sales drivers
•	Uncovered critical insights including diesel-to-hybrid shift post-2020 and mileage impact on resale value (30% correlation)
•	Delivered actionable recommendations for inventory optimization and targeted marketing strategies to boost profitability
